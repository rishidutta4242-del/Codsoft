# 🎬 Movie Rating Prediction

## 📌 Overview

Movie Rating Prediction is a Machine Learning project that predicts the rating of a movie based on its features, such as genre, director, actors, duration, and other attributes. The project aims to analyze historical movie data and build a regression model capable of estimating movie ratings with high accuracy.

This project demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis, feature engineering, model training, and evaluation.

---

## 🎯 Objective

The primary objective of this project is to develop a machine learning model that can accurately predict movie ratings based on various movie characteristics. It also helps in understanding the factors that influence audience and critic ratings.

---

## 🚀 Features

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Handling missing values
- Feature engineering
- Encoding categorical variables
- Training regression models
- Model performance evaluation
- Movie rating prediction

---

## 🛠️ Technologies Used

---

## 📂 Dataset

The dataset contains information about movies, including:

- Movie Name
- Genre
- Director
- Actors
- Duration
- Language
- Number of Votes
- Movie Rating (Target Variable)

---

## ⚙️ Project Workflow

1. Import the dataset.
2. Perform data cleaning and preprocessing.
3. Handle missing values.
4. Encode categorical features.
5. Perform exploratory data analysis.
6. Split the dataset into training and testing sets.
7. Train regression models.
8. Evaluate model performance.
9. Predict movie ratings.

---

## 📊 Model Evaluation

The model is evaluated using the following metrics:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

---

## 📈 Results

The trained regression model successfully predicts movie ratings by learning patterns from historical movie data. Model performance is evaluated using standard regression metrics to ensure reliable and accurate predictions.
