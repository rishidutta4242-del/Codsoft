# Credit Card Fraud Detection

## Overview
A Machine Learning project that detects fraudulent credit card transactions using a **Random Forest Classifier**. The dataset is preprocessed, balanced using **SMOTE**, and evaluated with appropriate classification metrics.

## Technologies Used
- Python
- Pandas & NumPy
- Scikit-learn
- Imbalanced-learn (SMOTE)
- Matplotlib & Seaborn

## Workflow
- Data preprocessing and normalization
- Handle class imbalance with SMOTE
- Train a Random Forest model
- Evaluate using Precision, Recall, F1-Score, and Confusion Matrix

## Results
The model successfully identifies fraudulent transactions with strong classification performance.
